CRUD
Criar o projeto
Criar o banco de dados
Criar a migração das tabelas - php artisan migrate


Criando as telas de Autenticação(LOGIN) siga os passos abaixo:
 - Verificar se possui o node instalado na maquina
 - composer require laravel/ui
 - php artisan ui vue --auth
 - npm install
 - npm run dev

Realizar teste com cadastro e login de usuário

Criar um grupo de rota e trasnferir as existentes para esse grupo

Criar dentro da pasta views um pasta chamada usuarios  e o arquivo list.blade.php ela irá listar os usuarios
Criar a rota de lista usuarios
Criar a controllers do usuarios - php artisan make:controller UsuariosController 
Criar a model Usuario - php artisan make:model Usuario
Criar a tabela usuarios - php artisan make:migration create_usuarios_table --create=usuarios

Na model Usuario definir os campos de preenchimento que serão utilizados
Criar em UsuariosController a função index
Usar o foreach em list.blade
